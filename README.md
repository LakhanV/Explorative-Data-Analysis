# Renewable Energy Interactive Visualization

This project demonstrates the use of interactive visualizations for renewable energy data using Plotly.

## Visualizations Included
1. Renewable Energy Share by Country (Choropleth Map)
2. Solar Energy Production by Country (Bar Chart)
3. GDP vs Renewable Energy Share (Scatter Plot)
4. Renewable Energy Trends Over Time (Line Chart)

## Technologies Used
- Python
- Pandas
- Plotly

## Dataset
Synthetic renewable energy dataset generated using Python.
